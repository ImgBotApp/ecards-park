# Locally

Run server

```
node app.js
```

# Heroku

Login

```
heroku login
```

Deploying code

```
git push heroku master
````

Ensure that at least one instance of the app is running

```
heroku ps:scale web=1
````

Visit the app at the URL generated by its app name

```
heroku open
```

View logs

```
heroku logs --tail
```
